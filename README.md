 AI-Career-Mentor

** Project Overview**

AI Career Mentor is a beginner-level LLM-powered application built using Python and OpenRouter API.

The system extracts student profile information, analyzes current skills, and generates personalized career guidance, learning roadmaps, and project recommendations.

 **Problem Statement**

Many students struggle to identify the next skills, projects, and learning paths required to achieve their target career goals.

This project demonstrates how Large Language Models (LLMs) can provide personalized career guidance.



 **Features**

- Extract student information into structured JSON
- Generate personalized career roadmaps
- Recommend learning resources
- Suggest beginner-friendly projects
- Maintain conversation history



 **Technologies Used**

- Python
- OpenRouter API
- DeepSeek V4 Flash
- JSON
- Requests Library
- Google Colab



 **Workflow**

1. Collect student profile information
2. Convert profile into structured JSON
3. Send data to LLM
4. Generate career roadmap
5. Generate project recommendations
6. Display personalized guidance



 **Sample Output**

- Recommended skills
- 3-month learning roadmap
- Project suggestions
- Career guidance
